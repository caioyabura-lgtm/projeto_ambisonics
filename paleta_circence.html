Paleta Circense

Estudo de cores para uma versão infantil do projeto, combinando o imaginário do circo com a linguagem do planetário, do som espacial e da instalação imersiva. A paleta busca alegria, contraste, ludicidade e boa leitura visual para web, cartazes, foyer e materiais de mediação.

Paleta principal

A paleta-base usa cinco cores para representar a atmosfera do circo: azul-petróleo para profundidade, bege de lona para acolhimento, dourado para brilho, terracota para energia e roxo profundo para o ambiente noturno do planetário.

Uso conceitual

O circo entra como linguagem de encantamento, entrada no imaginário e convite para crianças. O planetário entra como céu, noite azul, domo e escala. A união das duas referências produz uma estética de picadeiro cósmico.

Uso prático

As cores mais fortes devem aparecer em chamadas, botões e títulos. As cores claras servem para fundos, textos de apoio e peças educativas. O roxo/noite azul protege a identidade imersiva do planetário.

Aplicações no espetáculo infantil

As combinações abaixo ajudam a definir onde cada cor entra no projeto visual.

Domo e noite azul

Azuis profundos para manter a sensação de céu, domo, mistério e imersão tecnológica.

Chamadas infantis

Vermelho e dourado para botões, títulos, cartazes e momentos de alta energia.

Mediação educativa

Dourado claro e terracota suave para fichas, textos, atividades, sinalização e materiais de acolhimento.
